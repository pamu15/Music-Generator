# Music Generator 🎵

A Python project that generates piano music using deep learning (LSTM). The model is trained on MIDI files and can create new musical sequences automatically.  

---

##Features

- Processes and extracts notes from MIDI files.
- Trains an LSTM neural network on sequences of notes.
- Generates new music in MIDI format.
- Easy to run and expand with more MIDI data.

---

## 🛠️ Technologies Used

- **Python 3.11**
- **TensorFlow / Keras**
- **NumPy**
- **music21** (for MIDI processing)
- **VS Code** for development
